

# MPU6050 Sensor Monitoring with Arduino Uno (Wokwi Simulation)

## 📌 Project Overview

This project simulates an **MPU6050 accelerometer and gyroscope sensor** connected to an **Arduino Uno** using the **Wokwi simulator** within VS Code with PlatformIO. The system reads real-time orientation data and displays it through the serial monitor.

## 🎯 Features

- ✅ Simulated MPU6050 sensor data acquisition
- ✅ Real-time accelerometer (X, Y, Z axes) readings
- ✅ Real-time gyroscope (X, Y, Z axes) readings
- ✅ Serial monitor output for data visualization
- ✅ PlatformIO build system integration
- ✅ Wokwi virtual simulation (no hardware required)

## 🚀 Getting Started

### 1. Open the Project in VS Code

```bash
code .
```

### 2. Build the Project

```bash
pio run
```

Or use the PlatformIO toolbar: **Build → Build**

### 3. Run the Simulation

- Press `F1` and type **"Wokwi: Start Simulator"**
- Or click the **Wokwi icon** in the VS Code sidebar

### 4. View Serial Output

- The **Wokwi simulation panel** will automatically open
- Serial monitor will display sensor readings
- Example output:

```
MPU6050 Simulation Started
Accel X: 0.12  Accel Y: -0.03  Accel Z: 9.81
Gyro X: 0.01   Gyro Y: -0.02   Gyro Z: 0.00
```

```ini
framework = arduino
monitor_speed = 9600
```

## 🔬 Simulated vs Real Hardware

| Aspect | Simulation (Wokwi) | Real Hardware |
|--------|-------------------|---------------|
| MPU6050 Sensor | ✅ Virtual emulation | ✅ Physical sensor |
| Arduino Uno | ✅ Virtual emulation | ✅ Physical board |
| Serial Output | ✅ Wokwi Serial Monitor | ✅ USB Serial Monitor |
| Cloud Database | ❌ Not supported | ❌ Uno lacks Wi-Fi |
| Dashboard Graphs | ❌ Not supported | ❌ Requires additional modules |

> **Note:** Arduino Uno has **no built-in Wi-Fi**. For cloud/dashboard features, you would need an ESP32 or add an external Wi-Fi module (ESP8266, Ethernet shield).

## 🐛 Troubleshooting

### Issue: `pio device monitor` fails
**Solution:** Do not use `pio device monitor`. Use Wokwi's built-in serial monitor instead.

### Issue: No serial output in Wokwi
**Solution:** Ensure `Serial.begin(9600)` is in `setup()`.

### Issue: Build fails
**Solution:** Run `pio update` and ensure all dependencies are installed.
